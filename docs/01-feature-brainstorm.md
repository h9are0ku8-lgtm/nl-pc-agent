# 機能案壁打ち — 自然言語PCエージェント

日付: 2026-07-10  
プロジェクト: `original-tool`（nl-pc-agent）

## やりたいこと（理想）
自然言語でPCを操作するデスクトップエージェント。  
例: フォルダ整理 / PDF要約→Excel / Web取得→CSV / ファイル移動 / メール添付処理

## 実現可能性

| 機能 | MVP | 本格版 |
|---|---|---|
| ファイル整理・移動・リネーム | ◎ 今回実装 | ◎ |
| PDF→テキスト/Excel抜粋 | ◎ 今回実装 | ◎（要約精度向上） |
| CSV保存 | ◎ | ◎ |
| Webスクレイピング | ×（次） | Playwright等 |
| Slack連携 | ×（次） | Slack API |
| メール添付処理 | ×（次） | OS連携が必要 |
| 画面GUI自動操作 | × | Computer Use系 |

## 方針
- **ローカルCLIエージェント**として先に動かす（Electron化は後）
- 操作範囲は `workspace/` に制限（安全）
- 実行前にプラン確認
- Geminiがあれば高度な分解、無ければルールプラン
