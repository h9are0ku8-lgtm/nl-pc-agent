# システム・ツール記事（下書き）— 自然言語PCエージェント

## タイトル案
「ダウンロードフォルダを整理して」と言うだけで動く、安全なPCエージェントMVP

## 導入
面倒なPC作業を、自然言語で頼みたい。  
ただしOS全体を自由に触るAIは危険なので、**作業フォルダ限定・確認付き**のMVPを作った。

## できること
- inbox整理（拡張子ごと）
- ファイル移動・リネーム
- PDF→テキスト / Excel抜粋
- CSV保存

## まだできないこと
- 任意サイトの売上スクレイピング
- メール・Slackの本格連携
- 画面クリック操作

## 使い方
```bash
cd original-tool
npm install
npm start -- "inboxを整理して"
```

## 学び
- 理想のデスクトップエージェントは段階実装が必要
- 最初に「触ってよい場所」を決めると安全
- AIが無くてもルールプランで価値を出せる
