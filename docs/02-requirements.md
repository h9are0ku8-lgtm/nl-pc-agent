# 要件定義書 — 自然言語PCエージェント（MVP）

| 項目 | 内容 |
|---|---|
| 版 | 0.1 MVP |
| 作成日 | 2026-07-10 |
| 目的 | 自然言語の依頼をタスク分解し、安全な範囲でPCファイル作業を自動実行する |

---

## 1. 課題
ファイル整理やPDFの抜き出しなど、手順は単純でも手作業が多い。  
毎回同じ操作を繰り返したくない。

## 2. MVPスコープ

### できること
- 自然言語で依頼を受け取る
- タスクをステップに分解して表示
- ユーザー確認後に実行
- ファイル一覧 / 作成 / 移動 / コピー / リネーム
- 拡張子ごとの自動整理
- 直近更新ファイルの一括移動（slackフォルダなど）
- PDFテキスト抽出、PDF要約のExcel化（Gemini任意）
- 公開URLの表/JSONをCSV保存
- CSV / テキスト書き出し

### まだやらないこと
- ログイン必須サイトの巡回
- 実メールクライアント操作
- Slack API送信
- OS全体のGUI自動操作
- ワークスペース外の無制限操作

---

## 3. 安全要件
- 既定の操作ルートは `./workspace`
- ルート外パスは拒否
- 実行前にプラン確認（`--yes` で省略可）
- 危険なシェル任意実行は提供しない

---

## 4. 構成

```
ユーザー自然言語
  → planner (Gemini or rules)
  → 確認
  → tools (fs / pdf / excel)
  → workspace/inbox|outbox|slack
```
